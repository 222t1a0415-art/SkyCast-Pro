# SkyCast Pro

A professional weather dashboard built using HTML, CSS, and JavaScript.

## Features

- Current Weather
- 5-Day Forecast
- Geolocation Weather
- Search History
- Dark/Light Mode
- Celsius/Fahrenheit Toggle
- Responsive Design
- Local Storage

## Technologies

- HTML5
- CSS3
- JavaScript ES6
- OpenWeatherMap API
